Project 2

Flipkart Clone

A flipkart clone static page is a website that mimics the appearance and functionality of the popular e-commerce platform Flipkart. It is typically created for learning or practice purposes, and may include similar features such as a search bar, product listings, and a shopping cart. The design and layout of the page may closely resemble Flipkart's branding, with similar color schemes and fonts. However, a clone page does not have actual functionality to process transactions or manage inventory. Overall, creating a flipkart clone static page can be a useful exercise in practicing HTML and CSS skills and understanding the basics of web development.

Project 3

Counter App

The counter app is a simple web application that allows users to increment or decrement a counter value by clicking on buttons. The current value of the counter is displayed on the screen, and users can reset the counter to its initial value if needed. The app is a good example of how HTML, CSS, and JavaScript can be used together to create a responsive and interactive user interface.
